# RL Algorithms

# Implementation List
### Algorithm

- Deep Q Network (DQN)
- Double DQN
- Dueling DQN
- C51
- QR-DQN
- Soft Actor Critic (SAC)

### Environment 

- Cartpole
- Pendulum 
- Atari 
- ML-Agents 



# Install

```
 $ git clone https://github.kakaocorp.com/leonard-q/RL_Algorithms.git  
 $ pip install -r requirements.txt  
 $ python main.py 
```
